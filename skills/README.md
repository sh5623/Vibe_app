# Vibe App AI Skills System

이 폴더는 AI 에이전트(Antigravity)가 이 프로젝트에서 수행해야 할 **특수 작업 지침**이나 **자주 사용하는 명령어 조합**을 저장하는 곳입니다.

## 💡 사용 방법

1. **지침 저장**: 반복적으로 요청하기 번거로운 작업 지침(예: 특정 커밋 스타일, 배포 절차, 컴포넌트 구조 등)을 `.md` 파일로 작성하여 이 폴더에 넣습니다.
2. **스킬 호출**: 저에게 다음과 같이 요청하세요:
   > "skills 폴더의 `commit_guide.md` 읽고 작업해줘"
   > "새로운 스킬을 `deploy_skill.md`로 만들어줘"

3. **자동 인식**: 저는 작업을 시작할 때 이 폴더를 확인하여 프로젝트의 특수한 환경이나 규칙을 미리 파악할 수 있습니다.

## 📂 파일 구조 예시
- `commit_guide.md`: 이 프로젝트만의 커밋 메시지 규칙 및 절차
- `component_template.md`: 새로운 UI 컴포넌트 생성 시 지켜야 할 스타일 가이드
- `troubleshooting.md`: 자주 발생하는 에러와 해결 방법
