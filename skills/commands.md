# Custom AI Commands Mapping

이 파일은 채팅창에서 사용할 수 있는 사용자 정의 명령어와 그에 따른 AI의 행동 지침을 정의합니다.

| 명령어 | 매핑되는 행동 및 스킬 | 설명 |
| :--- | :--- | :--- |
| `/commit` | `skills/commit_guide.md` 읽기 및 실행 | 프로젝트 규칙에 맞는 커밋 수행 (main 브랜치 체크 포함) |
| `/fix-mobile` | `src/app/dev/styled.ts` 내 모바일 쿼리 최적화 | 모바일 짤림 현상이나 레이아웃 오류 자동 점검 및 수정 |
| `/vibe-check` | `GEMINI.md` 가이드라인 준수 확인 | 색상, 그라데이션, 글래스모피즘 등 디자인 퀄리티 체크 |
| `/status` | `git status` 및 `git branch` 요약 | 현재 작업 중인 브랜치와 변경 내용 요약 보고 |

## 💡 사용 가이드
- 사용자가 위 명령어 중 하나를 입력하면, AI는 즉시 해당 칸의 "행동 및 스킬"을 수행합니다.
- 새로운 명령어가 필요한 경우 이 파일에 행을 추가하여 정의할 수 있습니다.
